# **Predicción de riesgo de diabetes usando datos clínicos**
Este proyecto uliza técnicas de machine learning para predecir la probabilidad de diabetes a partir de variables clínicas.
## **Objetivo**
Desarrollar un modelo que identifique pacientes con riesgo de diabetes.
## Herramientas
Python
Pandas
scikit-learn
## Resultados
Modelo de regresión logística
Predicción de riesgo
Evaluación del modelo
## Conclusión
El modelo permite identificar pacientes con mayor riesgo, apoyando estrategias de prevención de salud.
